04
Clean Architecture

Tổ chức mã nguồn

Tổ chức mã nguồn theo chuẩn Modularization.

backend/
├── server.js # Entry Point
└── src/
    ├── config/ # Configs
    ├── controllers/ # Logic
    ├── middlewares/ # Interceptors
    ├── models/ # Database
    └── routes/ # Endpoints

Components

  • server.js: Khởi tạo App, kết nối Server & Socket.io.
  • config/: Cấu hình Database, biến môi trường (Env).
  • controllers/: Nhận Request, xử lý nghiệp vụ, trả Response.
  • middlewares/: Lớp trung gian (Check Token, Validate Input).
  • models/: Định nghĩa cấu trúc bảng (User, Course) qua ORM.
  • routes/: Định tuyến URL (VD: GET /api/users trỏ về đâu?).
